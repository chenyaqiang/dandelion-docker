dandelion-docker
================

Base image of the Dandelion project.

The environment is based on CentOS 6.4 and contains:

 * OpenJDK 6
 * Maven 3.2.1 
 * PhantomJS 1.9.7
 * OpenSSH Server (port 22 exposed)
 * Supervisor

The following users are also configured:

 * root/dandelion
 * dandelion/dandelion
